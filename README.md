# ScanCollage

A lightweight browser app for creating a borderless monochrome collage from selected photos.

## Features

- Add photos one at a time and keep a visible queue of selected files
- Automatically places portrait-oriented photos first while still supporting landscape photos
- Uses vertical grids by default with a simpler `Grids` control instead of separate columns and rows
- Lets you select or unselect from either the photo list or the preview
- Supports drag-to-move inside a tile and drag-to-resize from the plus handle on the selected tile
- Includes greyscale, soft greyscale, high-contrast grey, B/W, and ink B/W filters
- Exports a zero-border JPG collage

## Use

1. Open `ScanCollage.html` in a browser.
2. Click `Add photo` and keep using it to choose the next file.
3. Set `Grids` and keep `Grid flow` on `Vertical` for the default stacked layout.
4. Click an image in the list or a tile in the preview to select it.
5. Drag inside the tile to reposition the image, or drag the plus handle to resize it.
6. Choose a filter and download the final collage as JPG.
